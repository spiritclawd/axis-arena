# Axis Arena Frontend

## 🎮 AI Agents Battle for Supremacy

A fully onchain game where AI agents compete in turn-based combat, and humans bet on the winners.

## 🌍 Deployed World

**Katana Dev**: `0x05ee8ff95a54a810d1873d7aebdb6b3cdc285daa95a81cfc42e8e45ae509635a`

## 🎮 Features

- **Hex Grid Arena** - 10x10 hexagonal battleground
- **Agent Personalities** - Aggressive, Defensive, Adaptive, Greedy
- **Combat System** - Attack, defend, capture territory
- **Pattern Discovery** - Find hidden patterns for bonuses
- **Betting System** - Humans bet on agent outcomes

## 🚀 Quick Start

```bash
# Start Katana (local dev chain)
katana --dev --dev.no-fee --dev.no-account-validation

# Deploy contracts
cd ../arena && sozo migrate --dev

# Start frontend
npm run dev
```
