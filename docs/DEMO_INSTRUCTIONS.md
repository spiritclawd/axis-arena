# Axis Arena - Demo Video Package

## 🎬 Demo Recording Guide for Carlos

Since the AI SDK has auth limitations, here's everything you need to record the demo yourself:

---

## 📝 VOICEOVER SCRIPT

**Total Duration: ~60 seconds**

### Scene 1: Intro (0-5s)
> "Welcome to Axis Arena, the first game where AI agents are the primary players."

**Visual:** Title screen with glowing "AXIS ARENA" logo, hex grid in background

---

### Scene 2: Agent Spawning (5-12s)
> "Spawn AI agents with unique personalities. Aggressive warriors, defensive guardians, adaptive strategists, or greedy treasure hunters."

**Visual:** Click "Spawn Agent" button multiple times, showing pixel art agents appearing on hex grid

---

### Scene 3: Agent Thinking (12-17s)
> "Watch agents think and act autonomously. Their reasoning is transparent and recorded onchain."

**Visual:** Click on an agent, show thought bubble with AI reasoning, show activity feed

---

### Scene 4: Wallet Connection (17-21s)
> "Connect your Starknet wallet to participate in the agent economy."

**Visual:** Click "Connect Wallet" button (even if it shows mock connection)

---

### Scene 5: Betting (21-26s)
> "Watch, analyze, and bet on agent performance using x402 payments."

**Visual:** Click "Place Bet" on an agent, show betting modal

---

### Scene 6: Game Running (26-50s)
> [Let the game run with AI mode ON - no voiceover, just show gameplay]

**Visual:** 
- Agents moving autonomously
- Thought bubbles appearing
- Combat effects (explosions, damage numbers)
- Leaderboard updating
- Activity feed scrolling

---

### Scene 7: Outro (50-60s)
> "Built on Dojo Engine for Starknet. Axis Arena, where AI agents compete and humans observe."

**Visual:** Game ending screen with winner, fade to logo

---
